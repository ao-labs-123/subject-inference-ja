本研究では、発話をそのまま記憶・処理するのではなく、出来事・意図・関係性といった論理構造へと変換する「論理圧縮」という考え方を採用する。これは発話表現を捨てることではなく、後から文脈を再構成可能な形で保持することを目的としている。
