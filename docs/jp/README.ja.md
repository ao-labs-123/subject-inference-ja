## 概要
本リポジトリは、対話AIにおける誤読の原因となる
「早期の意味確定」を防ぐための確認機構を提案・実装するものです。
Stage 2 では意味推論は行わず、主語不明・曖昧性に対する
最小限の聞き返しのみを行います。
