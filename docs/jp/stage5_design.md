# Stage 5：名詞同士の格関係の明示化

## 概要

Stage 5 では、文中に登場する**名詞同士の関係（誰が・何を・誰に・どの立場で）**を明示的に扱う。
助詞や語順だけに依存せず、意味役割（格）に基づいて文の構造を解釈することで、誤読を根本的に防ぐ。

## 課題

従来の対話AIでは、以下のような問題が生じやすい。
	
	•	助詞の省略や曖昧な表現に弱い
	•	名詞が多い文で、主語・目的語・対象の取り違えが起きる
	•	文法的には正しいが、意味的に不自然な解釈をしてしまう

例：

「先生に生徒のレポートを渡した」

→ 誰が渡したのか / 誰のレポートなのかが曖昧。

## 改善案
文を単なる語の並びとして扱うのではなく、
各名詞に 意味役割（Semantic Roles） を付与する。

	•	行為者（Agent）
	•	対象（Patient / Object）
	•	受益者（Recipient）
	•	所有関係（Possessor）
	•	場所・時間などの補助的役割

これにより、
	
	•	名詞間の関係性を構造として保持
	•	文脈に基づく一貫した解釈
が可能になる。

## Stage 5 の位置づけ
Stage 1〜4 で
	
	•	誤読の検出
	•	必要最低限の聞き返し
	•	因果関係推定
	•	修飾構造の整理

を行った上で、最終的に意味構造を確定させる段階が Stage 5 である。

## 効果
	•	長文・複雑な文でも誤解が起きにくい
	•	対話AI・音声AI・翻訳・IME などへの応用が可能
	•	「文法的に正しいが意味的に間違っている」解釈を防止
