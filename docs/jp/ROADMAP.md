#  プロジェクト・ロードマップ

本ロードマップは、「誤読しない対話AI」を中心とした一連の技術群  
（IME 改良、多言語翻訳、安全検知、日常利用AIなど）を段階的に開発していくための計画を示します。

透明性・再現性・国際的な貢献を重視し、明確なフェーズ構造で進行します。

---

##  フェーズ1 — 誤読しない対話AIの核（現在のフェーズ）
**目的:** 人間とAIの意思疎通における「誤解」を構造的に防止する基盤エンジンを構築する。

### タスク
- ルールベース＋確率的解釈フィルタの設計  
- 文脈に基づく曖昧性の排除  
- 会話状態追跡モデルの実装  
- ベースライン実装の公開  
- API 仕様書の公開

**ステータス:**  進行中

---

##  フェーズ2 — 多言語対応（Multilingual Expansion）
**目的:** 複数言語の解釈ズレを最小化し、文脈保持型の多言語対話を実現する。

### タスク
- 日英間の精度最適化  
- 対応言語の拡張  
- 言語非依存の意味表現を導入  
- 文脈を保持した翻訳の整合性向上

**ステータス:** 計画中

---

##  フェーズ3 — 入力システム統合（IME モジュール）
**目的:** 誤読防止を “入力段階” から行う新しい IME を設計する。

### タスク
- 文脈に沿う IME 予測モデル  
- 意図しない変換候補の抑制  
- 多言語タイピング対応  
- 外部アプリ向けの IME API 提供

**ステータス:** 計画中

---

##  フェーズ4 — 安全・詐欺・悪意検知モジュール
**目的:** インターネット上の以下を検出するシステムを構築する：  
- 誹謗中傷  
- 詐欺パターン  
- 偽情報  
- ソーシャルエンジニアリング  
- 悪意ある文章変換

### タスク
- リアルタイム検知エンジン  
- ブラウザ／SNS 拡張プロトタイプ  
- 開発者向けパッケージ  
- 公開データセットとベンチマーク

**ステータス:** 計画中

---

##  フェーズ5 — 日常利用AIモジュール
**目的:** 誤読防止エンジンを “日常生活ツール” に落とし込む。

### モジュール例
- 誤読しないチャットアシスタント  
- 文脈保持翻訳システム  
- 緊急時の状況説明サポートAI  
- 文章要約・書き換え・質問応答  
- 家庭デバイスとの統合（任意）

**ステータス:** 計画中

---

##  フェーズ6 — コミュニティ & オープン開発
**目的:** 世界中の開発者が参加できる開発生態系を作る。

### タスク
- Contribution Guide の整備  
- API ドキュメントの公開  
- サンプルアプリの提供  
- Issue テンプレート／議論板の追加  
- 多言語ドキュメント（日本語＋英語）の公開

**ステータス:** 計画中

---

## ✔ ビジョン
- 誤解されない  
- 多言語で意味が崩れない  
- 安全性が確保されている  
- 誰でも開発に参加できる  
- 世界の日常生活で自然に使われる

そんな対話AIエコシステムの構築を目指します。
